# MixerChatApp
ミキサーのコメントをいい感じに表示するやつ

# 使い方
ユーザー名を入力して接続を押すとそのチャンネルのコメントを取得します。

# 機能
棒読みちゃんに送る  
コメントを書き込む

# 実装したい機能
OBSや棒読みちゃんを同時に立ち上げる機能
